# Plan 07-02: G8 Phone-Native Streamed-Corpus Repair

## Sovereign Gate

Promote nothing unless the REDMAGIC NX789J runtime starts from phone-streamed text/token cache plus immutable Gemma assets, generates layer0 input and layer0/layer1 PLE tensors on the phone, runs two real OpenCL layers, performs the rank-4 adapter backward/update on the phone, writes a replayable checkpoint, and passes RunPod PyTorch falsifiers. G1 and G3 remain regression gates.

## Tasks

1. Extend the G7 cache ABI with `labels.u32.bin`, `loss_mask.u8.bin`, `position_ids.u32.bin`, and manifest hashes.
2. Export frozen Gemma assets needed for phone token-to-hidden generation: full main embeddings, layer0/1 PLE token slices, layer0/1 PLE projection slices, and PLE norm weights.
3. Add the minimal runtime bridge: read token cache and frozen BF16 assets on phone, produce `layer_input`, `per_layer_input_layer0`, and `per_layer_input_layer1` without consuming RunPod hidden fixtures.
4. Wire the bridge into OpenCL layer0/layer1 execution and reuse the existing adapter SGD kernels against phone-generated `h0` and phone-generated stop-gradient `h1`.
5. Emit telemetry, artifact manifests, replay manifests, checkpoint manifests, trainable hashes, and frozen pre/post hashes.
6. Generate a RunPod PyTorch oracle for the same token cache and compare token cache, token-to-hidden tensors, layer0/layer1 outputs, gradients, and checkpoint update tensors.
7. Run G1/G3 regressions after the G8 runtime changes.
8. Run falsifiers; if any critical falsifier fails, keep Phase 07 failed and write next tasks instead of promoting.

## Falsifiers

- Fail if any runtime input includes RunPod/Mac `layer_input`, `per_layer_input`, `layer0_output`, or `layer1_output` fixtures.
- Fail if token IDs, masks, labels, loss masks, or positions mismatch the HF tokenizer reference.
- Fail if PLE layer1 is generated from layer0 output instead of original token IDs and main embeddings.
- Fail if layer or adapter comparisons fall below the existing cosine thresholds or contain non-finite values.
- Fail if frozen base hashes change, checkpoint files are missing, or replay metadata is incomplete.
