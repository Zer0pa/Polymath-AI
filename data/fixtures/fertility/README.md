# Tokenizer fertility fixtures

Tiny text fixtures used by Phase 0F (Experiment 1) tokenizer fertility audit
on the host machine, before the bulk Seed Corpus v0 dataset is available
through Hugging Face.

Each fixture is a short, public-domain or CC0 sample of natural text.
Sources: Universal Declaration of Human Rights (1948 — public domain in
most jurisdictions; the official multilingual translations are widely
distributed for educational use).

The full Phase 0F audit runs against per-language slices of the Seed Corpus
v0 dataset on Hugging Face, not these fixtures. The fixtures exist to
exercise the fertility plumbing on the host before the phone attaches.
